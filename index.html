A Thought-Provoking Title

An optional but useful blurb to display an abstract in a couple of sentences. Maybe include a nice image? It could span the width of this page.

Getting Started

Use the template repository to 'Create a new repository'. Use your favourite web server to serve the files in the repository. Make changes to index.html to create your article.

This is a template for presenting technical content through an interactive medium. A large amount of the work used here has been taken from the Distill journal. Release v2.8.0 has been used here.

Citations

After adding a Bibtex bibliography entry, citations can be inlined. At the end of the article a bibliography will be added.

Use <d-cite key="conlen2019launching"></d-cite> to make a citation and add the citation to the bibiliography as: <d-bibliography><script type="text/json"> [ ["conlen2019launching", // <-- this is used in the 'key' field {"title":"Launching the Parametric Press", "author":"Conlen, Matthew and Hohman, Fred", "journal":"Visualization for Communication (VisComm) at IEEE VIS", "year":"2019", "url":"https://parametric.press", "type":"article"}], ["another2020ref", {"title":"Another Title", "author":"Author, Another", "journal":"Another Title for Another Paper", "year":"2020", "url":"https://distill.pub", "type":"article"}] ] </script></d-bibliography>

As per the Distill Guide, try to add the URL where possible to allow for linking.

Footnotes

Footnotes can be added like thisThis is a footnote!. <d-footnote>This is a footnote!</d-footnote>

These will also be added to the end of the article with the bibliography.

<pre class="mermaid"> graph LR A[Christmas] -->|Get money| B(Go shopping) B --> C{Let me think} C -->|One| D[Laptop] C -->|Two| E[iPhone] C -->|Three| F[Car] subgraph Devices D E F end </pre>

or a State Chart

    %%{init: {'theme':'forest'}}%%
    stateDiagram-v2
      [*] --> Still
      Still --> [*]
      Still --> Moving
      Moving --> Still
      Moving --> Crash
      Crash --> [*]

Change layout from graph LR to graph TD for a vertical graph. See MermaidJS diagram syntax for more examples and diagram types.

Plotly Graphs

Plotly charts can also be used

To export, build the Plotly figure and then save. Whilst other Plotly themes/templates can be used, it would be a good idea to use one that goes well with the theme of the rest of the article. simple_white is recommended.

# Normal plotly trace will also work import numpy as np import pandas as pd import plotly.express as px x = np.arange(100) y = np.sin(x) + np.random.rand(100) fig = px.line(df, x="x", y="y", template="simple_white") fig.write_html("line-chart-2.html", include_plotlyjs="cdn")

The exported HTML can be embedded in the page using an iframe:

<iframe frameBorder="0" class="l-page" src="charts/line-chart-2.html" height="300px"></iframe>

If required, the graph can spill out to use more horizontal space by adding class="l-page" to the tag.

The use of include_plotlyjs="cdn" when calling fig.write_html will not include the PlotlyJS library in the exported file (the default behaviour is to include it with each export). This will reduce the size of each export by ~3MB as the library is included by adding a call to the CDN. If this is not sufficient, the documentation has other options for exporting that may be suitable.

Images

Images can be added directly to the main section:

<figure> <img width="100%" src="images/moon.jpg"></img> <figcaption> Photo by CHUTTERSNAP on Unsplash. </figcaption> </figure>

or to the side:

<figure class="l-gutter"> <img width="100%" src="images/bender.jpg"></img> <figcaption> Photo by Francesco Tommasini on Unsplash. </figcaption> </figure>

Code Blocks

Code blocks use PrismJS under the hood. They can be inline, for example, to import numpy in Python: import numpy as np

Or in a separate block.

import numpy as np import pandas as pd df: pd.DataFrame = pd.read_csv("/path/to/csv")

which is generated using:

<d-code block language="python"> import numpy as np import pandas as pd df: pd.DataFrame = pd.read_csv("/path/to/csv") </d-code>
